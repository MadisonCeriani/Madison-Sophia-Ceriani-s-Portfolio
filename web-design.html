Artifact 01

NASA Space Explorer App

The NASA Space Explorer App is an example of API integration done effectively. The design is responsive and aligned with brand guidelines. I love space, and in my spare time I enjoy watching documentaries about our galaxy. Working with APIs is not as challenging as it may seem. Fetching data went well, and the modal window was effective. An if statement in JavaScript handled video entries with a clear clickable link so users could access content regardless of media type. A bug came up during development, but with help from GitHub Copilot I resolved it and delivered a smooth experience.

Artifact 02

Intel Sustainability Webpage

The Intel Sustainability page was one of my first projects during the Global Career Accelerator web development internship. The objective was to present Intel's sustainability timeline in a way that is compelling and consistent with brand identity. The most challenging aspect was responsiveness. In desktop view, cards are arranged horizontally; on mobile, they stack vertically for readability. Chrome DevTools helped identify repeated image sizing issues. I used an external tutorial to solve the issue and delivered a responsive page that guides users through Intel's timeline clearly.

Artifact 03

L'Oréal Product Aware Routine Builder

When designing this chatbot-driven project, JavaScript, HTML, and CSS were used during my internship in the Global Career Accelerator. The objective was to use an API to create a routine generator that helps users identify products that fit their needs. A Bootstrap-inspired modal window was added, and a search bar allows users to search for products and preserve selections after refresh. Most issues occurred while deploying with Cloudflare Workers. Securing the API was essential, and extra setup effort ensured private credentials remained protected in my codespace.

Artifact 04

Charity Water Game

The planning process was essential to designing the Charity Water Game. The objective was to create a game that encourages younger users to learn about the mission and get involved. This project had a clearly defined audience and succeeded despite development challenges. As part of the target age group, I contributed meaningful perspective to the design. The final product is responsive and includes audio, multiple difficulty levels, pause and reset controls, and visuals aligned with the client's brand identity. This project reinforced how even small design details influence communication and trust.
