Artifact 01

NASA Space Explorer App

The NASA Space Explorer App makes space content more accessible and engaging by presenting it in a clean, responsive gallery. Designed with both usability and brand consistency in mind, the gallery delivers a seamless experience across devices while staying aligned with NASA’s visual identity. My interest in space guided my design decisions prioritizing clarity and discovery, ensuring users can interact with different types of media effortlessly. By integrating NASA’s API effectively, the app dynamically presents content in a way that feels intuitive rather than overwhelming. Features like the modal view and the if statement in JavaScript handled video entries with a clear clickable link ensure that users can access information smoothly, regardless of format. While development included challenges, resolving them—such as debugging issues during implementation—resulted in a more polished and reliable experience. Overall, this project demonstrates how thoughtful design and technical decisions can make complex data more approachable and increase user engagement. .

Artifact 03

L'Oréal Product Aware Routine Builder

This project effectively utilizes an API to create a routine generator that helps users identify products tailored to their needs. Designed with a strong focus on usability, features include a Bootstrap-inspired modal window, and a searchable interface allowing users to easily explore products and revisit their selections. Issues occurred during the Cloudflare Worker deployment process, but resolving them ensured the security of the private API while maintaining functionality. Although my access to the API was temporary, this project still displays the importance of thoughtful front-end and backend design when creating a responsive user-friendly webpage.

Artifact 04

Charity Water Game

The Charity Water Game is designed to engage a young adult audience in understanding and supporting clean water initiatives through an interactive, accessible gaming experience. With the brand initiative present in the gameplay the project encourages users not just to learn about the mission, but to feel motivated to get involved. Features such as splash audio, multiple difficulty levels, pause, and reset controls create a flexible and inclusive experience, while visuals aligned with the client’s brand reinforce credibility and consistency. Overall, the final product displays how thoughtful design choices can reinforce communication, and support a meaningful cause. .
